# Periplus Shopping Cart Automation Test

Hi! In this repository, I have created an automation test using 
IntelliJ to verify the shopping cart functionality of a web-based 
application called Periplus. If you would like to try it, please 
clone this repository and replace the email and password with your 
own Periplus account credentials. Thank you!

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager

## How to Run
1. Clone this repository
2. Open the project in IntelliJ IDEA
3. Go to `src/test/java/PeriplusCartTest.java`
4. Replace `TRYEMAIL@GMAIL.COM`(Email
5. ) and `CHANGEHEREYA`(Paswword)with your Periplus account credentials
6. Run the test by right-clicking `PeriplusCartTest.java` → Run

## Test Scenario
- Open Google Chrome
- Navigate to https://www.periplus.com
- Login with registered account
- Search for a product
- Add product to cart
- Verify product successfully added to cart
